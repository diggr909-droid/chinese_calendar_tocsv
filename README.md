# 中文日历数据生成工具

一个用于生成包含公历、农历、节假日、节气等信息的中文日历数据工具，并输出为CSV文件。

## 功能特性

- 支持生成指定日期范围内的日历数据
- 包含公历和农历的详细信息
- 自动识别法定节假日和工作日
- 显示节气、生肖、干支纪年等中国传统文化元素
- 输出为UTF-8编码的CSV文件，支持直接用Excel打开
- 默认生成当前年份的完整日历数据

## 输出数据字段

| 字段名称 | 说明 |
|---------|------|
| 公历日期 | 完整的公历日期（YYYY-MM-DD） |
| 公历年 | 公历年 |
| 公历月 | 公历月 |
| 公历日 | 公历日 |
| 公历旬 | 当月的上、中、下旬 |
| 周数 | ISO周数 |
| 星期 | 星期几（周一至周日） |
| 农历年 | 农历年 |
| 农历月 | 农历月（数字） |
| 农历日 | 农历日（数字） |
| 是否闰月 | 该月是否为闰月 |
| 生肖 | 当年生肖 |
| 干支年 | 干支纪年 |
| 干支月 | 干支纪月 |
| 干支日 | 干支纪日 |
| 节气 | 当日节气（如立春、清明等） |
| 是否节假日 | 是否为法定节假日 |
| 是否工作日 | 是否为工作日 |
| 节日名称 | 节假日名称 |

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 基本用法

直接运行程序，默认生成当前年份的完整日历数据：

```bash
python main.py
```

程序会提示输入开始日期和结束日期，直接按回车使用默认值（当前年份的第一天和最后一天）。

### 指定日期范围

按照提示输入自定义的开始和结束日期（格式：YYYY-MM-DD）：

```
请输入开始日期（格式：YYYY-MM-DD，默认：2026-01-01）: 2026-01-01
请输入结束日期（格式：YYYY-MM-DD，默认：2026-12-31）: 2026-06-30
```

## 输出文件

生成的CSV文件会保存在`output`文件夹中，文件命名格式为：

```
[开始日期][结束日期]_chinese_calendar.csv
```

例如：`20260101-20261231_chinese_calendar.csv`

## 依赖库

- `lunar_python`：农历转换库
- `chinese_calendar`：中国法定节假日库
- Python标准库：`csv`、`os`、`datetime`

## 注意事项

- 目前`chinese_calendar`库支持的年份范围为2004年至2030年
- 程序会自动检查输入日期是否在支持范围内
- 输出文件使用UTF-8-BOM编码，确保Excel能正确显示中文
